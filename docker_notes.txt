W katalogu z docker-compose.yml:
docker compose up -d

Sprawdzenie:
docker ps

włączenie / wyłączenie (NAMES: mysql1, mysql2, mysql3, mysql4, mysql5):
docker stop mysql1
docker start mysql1


Sprawdzenie bazy z lini komend (hasło to: root)
docker exec -it mysql1 mysql -u root -p
USE app;
SELECT * FROM users;
